# 06 · 合规与信任框架

> ⚠️ 这是整个项目的**生命线**，不是可选项。这个生意建立在信任上，而信任的前提是合规和诚实。一次越线，可能既毁信誉又担法律责任。
>
> 注：本篇是认知和原则框架，**不是法律意见**。涉及具体经营、收费、跨境、加密等，务必咨询当地持牌律师/合规专业人士。

---

## 一、最大的红线：投资建议 / 投顾资质

在多数司法辖区，**提供个性化投资建议、推荐具体买卖、代客理财通常需要持牌**，无证从事可能违法。

### ✅ 安全区（信息 / 教育 / 研究定位）
- 分享"发生了什么事实"（某产品发布、某机制更新、某链上数据）。
- 科普"某个机制/玩法如何运作、有什么风险"。
- 分享你的**方法论和分析框架**。
- 客观呈现信息，揭示风险。
- 明确标注"不构成投资建议，自行决策、自负风险"。

### ❌ 危险区（投顾 / 喊单，避免）
- ❌ "买入 X""现在是入场点""目标价 Y"——个性化/指令性买卖建议。
- ❌ 预测具体资产价格涨跌。
- ❌ 承诺、暗示收益（"稳赚""翻倍""保本"）。
- ❌ 代客操盘、代为下单、托管资金。
- ❌ 收取业绩分成（按盈利抽成，更像受监管的资管）。

> **定位口诀：我提供"信息和分析"，你做"决策"。我说"发生了什么、风险是什么"，不说"你该买什么"。**

## 二、其他必须避开的红线

| 红线 | 说明 | 后果 |
|------|------|------|
| **夸大 / 虚构战绩** | 编造或美化收益记录 | 虚假宣传，SEC 类执法重点 |
| **抢帽子交易** | 自己先持仓，再推荐给订阅者拉高出货 | 操纵市场，违法 |
| **拉人头 / 层级返佣** | 靠发展下线、层层分销获利 | 涉嫌传销/非法集资 |
| **代币/盘子推广** | 收钱推广未披露的项目、参与坐庄 | 欺诈、操纵 |
| **荐股不披露利益** | 推荐自己有利益关系的标的而不声明 | 利益冲突，违规 |
| **无资质收款经营** | 不合规的收款/开票/跨境结算 | 税务/经营违规 |
| **数据来源侵权** | 抓取/转载违反对方条款或版权的内容 | 侵权风险 |

## 三、标准免责声明（每次交付都应包含）

> 以下为示例框架，**请根据你的实际业务和当地法律，请专业人士定稿**：

```
免责声明：
本内容仅为信息分享与研究交流，不构成任何投资、财务、法律或税务建议，
也不构成任何买卖要约或推荐。所有信息可能存在错误、延迟或不完整，
请自行独立核实。任何决策及其后果由您自行承担，本服务不对任何
盈亏负责。市场有风险，参与需谨慎，请勿投入超过您可承受损失的资金。
```

- 放在每期简报、社群说明、订阅页显著位置。
- 加密/金融相关内容尤其要醒目。

## 四、信任建设：把"诚实"做成护城河

在一个充满夸大的市场，**诚实是最稀缺、最难被复制的差异化**。具体做法：

### 1. 诚实的战绩记录（最重要）
- 维护一个**公开、可追溯的战绩追踪表**（模板见 `templates/战绩追踪表.md`）。
- **同时记录命中和踏空/看错**，不只晒赢。
- 每条机会标注当时的判断、时间戳、后续实际结果。
- 定期（如每月）公开复盘。
> **一个敢公开自己看错记录的人，比一个只晒盈利的人可信一百倍。** 这正是你打败 alpha 群的方式。

### 2. 透明的利益披露
- 如果你自己也参与了某个机会，**主动披露**："我自己也关注/参与了这个，存在利益关系。"
- 永远先披露，绝不利用订阅者给自己抬轿。

### 3. 克制与诚实的表达
- 不确定就说不确定。没好机会就说没有。
- 强调风险多于强调机会。
- 不制造 FOMO、不催促、不贩卖焦虑。

### 4. 兑现承诺
- 说每周发就每周发，说什么时候答疑就准时。
- 可靠性本身积累信任。

## 五、风险事件应对预案

| 风险事件 | 预案 |
|---------|------|
| 推荐关注的机会后续暴雷/归零 | 主动复盘、承认、分析教训（而非删帖装没事） |
| 信息出错 | 立即更正并公告，不掩盖 |
| 被质疑造假 | 因为有公开可追溯记录，能自证清白（所以记录要真） |
| 平台封号 | 因为有自有邮件名单，能转移订阅者（所以要拥有名单） |
| 监管问询 | 因为定位清晰（信息/教育）+ 有免责，风险可控；必要时咨询律师 |

## 六、合规自检清单（每次交付前过一遍）

- [ ] 内容是"信息/分析"而非"买卖指令"？
- [ ] 没有承诺或暗示收益？
- [ ] 有醒目的免责声明？
- [ ] 战绩记录真实、含失败案例？
- [ ] 涉及自身利益的已披露？
- [ ] 没有拉人头/返佣分层机制？
- [ ] 没有抓取/转载侵权内容？
- [ ] 收款/经营方式合规？

> 任一项不过关，**先停下，不发**。

---

## 七、核心心法

> **这个生意的全部资产是"信任"。信任的积累以年计，摧毁以分钟计。**
> 割韭菜的收益是一次性的、有法律风险的；诚实经营的复利是长期的、可持续的。
> 你在赛道分析里看到的那些被 SEC 起诉的 alpha 群和假 bot，就是反面教材。
> **做那个"在充满噪音和骗局的市场里，一直说真话、记录可查"的人——这本身就是最深的护城河。**
