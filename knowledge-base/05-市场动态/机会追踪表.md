# 机会追踪表

> 所有发现的机会都登记在这里，配合 `定期扫描SOP.md` 使用。这是你的"机会数据库"。
> 状态流转：待评估 → 观察 → 深入研究 → 验证中 → 执行中 / 已淘汰 / 已退场

---

## 一、使用说明

1. 日扫描发现的机会，先记到【待评估】，一句话即可。
2. 周扫描时用 `03-评估方法论/01-检查清单.md` 打分，更新状态。
3. 每个机会记录关键信息和打分，方便日后复盘。
4. **淘汰的机会不要删除**，移到【已淘汰归档】并写原因——这是认知积累。

---

