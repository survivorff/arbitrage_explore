# 00 · 导读与使用指南

> 在读任何具体内容之前，先花 5 分钟读完这一篇。它决定你能不能用对这个库。

---

## 一、这个库的核心信念

1. **套利不是投机，是生意的本质。** 任何把"低价的价值"搬到"高价的地方"的行为，都是套利。商人、中介、做市商、批发商、代购、量化基金，本质都在做同一件事：捕捉价差，同时提供流动性。

2. **赚钱 = 价差 − 摩擦，且价差归你独有的程度决定它能赚多久。** 这是贯穿全库的唯一公式。

3. **认知决定你能看到的机会。** 同一个市场，有人只看到混乱，有人看到结构。差别不在信息量，在框架。

4. **信息差是有半衰期的。** 越早拿到、越独占，越值钱。所以这个库不只是"学"，还要"持续扫描"。

---

## 二、怎么用这个库（三种模式）

### 模式 A：建立认知（一次性，约几小时）
按顺序读完 `01-认知基础` 和 `02-套利分类体系`。读完后你应该能做到：看到任何一个生意或现象，立刻说出它在套利哪个维度、价差从哪来、寿命多长。

### 模式 B：评估具体机会（每次遇到机会时）
打开 `03-机会评估方法论/01-机会评估检查清单.md`，逐项打分。低于阈值的直接放弃，避免浪费时间和钱。

### 模式 C：定期扫描市场（每周/每月）
按 `05-市场动态/定期扫描SOP.md` 跑一遍流程，把新发现的机会登记到 `机会追踪表.md`。这是你"站在信息前端"的引擎。

---

## 三、阅读这个库时的心态

- **不要急着找"那个能让我暴富的机会"。** 暴富机会的本质是"别人还没看到的结构性价差"，它只会出现在你有框架之后。
- **把每一个现实现象都拿来练习。** 看到楼下奶茶店、看到一条带货视频、看到一个跨所差价，都用框架拆一遍。练到形成肌肉记忆。
- **区分"信息"和"认知"。** 信息会过期，认知不会。先投资认知，再用认知去采集信息。

## 四、几个贯穿全库的关键术语

| 术语 | 含义 |
|------|------|
| **价差 (Spread)** | 同一份价值在不同维度上的价格差 |
| **摩擦 (Friction)** | 实现这笔套利要付出的所有成本（手续费、物流、时间、资金占用、损耗、风险） |
| **净价差** | 价差 − 摩擦，真正落到你口袋的部分 |
| **窗口 (Window)** | 机会存在的时间长度 |
| **收敛 (Convergence)** | 价差被竞争者抹平的过程 |
| **护城河 (Moat)** | 让价差长期归你独有的结构性壁垒 |
| **做市 (Market Making)** | 同时提供买卖报价、靠价差盈利、为市场提供流动性 |
| **半衰期 (Half-life)** | 一个信息差从产生到失效的时间 |
