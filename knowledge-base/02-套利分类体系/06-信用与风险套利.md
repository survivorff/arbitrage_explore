# 06 · 信用/风险套利（Credit & Risk Arbitrage）

> 替别人承担风险、提供信用背书，换取价差。金融业的根基，也是最高级的套利维度之一。

---

## 本质

有人有风险/资金需求，有人有承担能力。你介入，承担别人不愿/不能承担的风险，或提供别人缺乏的信用，赚取风险补偿。**你赚的价差本质是"风险的价格"。**

## 价差来源
- **风险偏好差异**：有人厌恶风险（愿付费转移），有人能管理风险（愿承担收费）。
- **信用差异**：有人有信用/抵押，有人没有，中间有价差（利差）。
- **信息+能力**：你能更好地评估和分散风险（保险精算、风控模型）。

## 典型场景

| 形态 | 说明 |
|------|------|
| 银行存贷 | 低息吸储、高息放贷，赚利差（最经典的信用套利） |
| 保险 | 收保费、承担赔付风险，靠大数定律和精算盈利 |
| 保理/供应链金融 | 垫资买应收账款，赚时间+风险溢价 |
| 二手车/二手房质保 | 承担"东西有问题"的风险，提供质保背书 |
| 担保/背书 | 用自己的信用为交易背书收费 |
| 加密借贷/做市 | 提供流动性承担无常损失/价格风险，赚利息+手续费 |
| Carry Trade | 借低息货币、投高息资产，赚息差（承担汇率风险） |

## 摩擦与风险
- **显性**：资金成本、坏账/赔付、合规成本。
- **隐性**：尾部风险（小概率大损失）、流动性风险、对手方违约。
- **特有风险**：**这是风险最致命的维度**。平时赚稳定小钱，但黑天鹅来时可能一次亏光（2008、各种暴雷、加密的连环清算）。"捡钢镚在压路机前面"——多数时候没事，出事就毁灭。

## 如何加护城河
1. **风控能力**：你的核心壁垒就是"比别人更会评估和管理风险"。这是数据、模型、经验的积累。
2. **分散化**：用大数定律把单笔风险摊薄（保险的本质）。
3. **资金成本优势**：能以更低成本拿到资金（牌照、规模、信用），利差就更大。
4. **对冲工具**：用衍生品对冲掉不想要的风险，只保留你能赚的那部分。

## 实操要点
- **核心心法：不要赚"看起来稳定、实则在累积尾部风险"的钱。** 很多信用套利的崩溃，都是因为长期忽略了小概率灾难。
- **永远问："最坏情况我亏多少？能承受吗？"** 风险套利的生死线是"活下来"，不是"赚得多"。
- **加密领域尤其危险**：脱锚、清算螺旋、智能合约漏洞、跑路，都是"看似无风险高息"背后的真实风险。**高息=高风险，没有例外。**
- 对个人/小团队：信用风险套利门槛高、风险大、常需牌照，**不建议作为起步赛道**，但要理解它（因为很多"高收益"陷阱本质是没被定价的风险）。
