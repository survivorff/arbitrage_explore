# 09 · 统计/概率套利（Statistical Arbitrage）

> 单笔不一定赚，但大量重复后，概率站在你这边。量化、做市、对冲的核心维度。

---

## 本质

不追求每一笔都赚，而是找到一个"长期期望值为正"的策略，靠**大量重复 + 大数定律**把概率优势兑现成稳定收益。赌场和保险公司就是这么赚钱的——单局可能输，长期必赢。

## 价差来源
- **统计规律**：价格的均值回归、相关性、历史规律。
- **技术/速度能力**：能比别人更快发现并执行微小价差（做市、高频）。
- **承担短期风险**：靠提供流动性赚买卖价差（做市商）。

## 典型场景

| 形态 | 说明 |
|------|------|
| 做市 (Market Making) | 同时挂买卖单，赚买卖价差(spread)，提供流动性 |
| 配对交易 (Pairs Trading) | 做多被低估、做空被高估的相关资产，赌价差回归 |
| 三角套利 | 加密里 A→B→C→A 的汇率闭环价差 |
| 高频/MEV | 极快捕捉微小价差（加密链上 MEV、CEX 高频） |
| 网格交易 | 在区间内自动低买高卖，靠波动赚钱 |
| 对冲基金策略 | 各种市场中性、统计套利策略 |

## 摩擦与风险
- **显性**：交易手续费（高频下手续费是头号成本）、基础设施成本（服务器、数据、低延迟）、技术研发。
- **隐性**：滑点、模型失效、策略拥挤（同样策略的人多了，价差消失）。
- **特有风险**：
  - **模型失效**：历史规律不代表未来，"均值回归"可能不回归（LTCM 之死）。
  - **速度军备竞赛**：高频/MEV 是红皇后赛跑，利润被基础设施投入吃掉。
  - **杠杆 + 尾部风险**：很多统计套利用杠杆放大微小价差，黑天鹅时爆仓。

## 如何加护城河
1. **技术/速度优势**：更低延迟、更快执行（但这是持续烧钱的军备竞赛）。
2. **独特策略/数据**：别人没有的策略或数据源（alpha 的真正来源）。
3. **成本优势**：更低手续费费率（做市商返佣、VIP 费率）让你在别人不赚的价差上还能赚。
4. **风控与资金管理**：在保证活下来的前提下持续运行（这往往比策略本身更重要）。

## 实操要点
- **这是最"工程化"的维度**，适合有技术能力的团队。核心竞争力是：策略 + 执行速度 + 风控 + 成本。
- **手续费费率几乎决定生死**：高频/做市场景下，能否拿到 maker 返佣、VIP 低费率，直接决定盈亏。
- **务必小资金验证**：统计套利的回测和实盘差距巨大（滑点、延迟、拥挤）。先小资金跑通，再放量。
- **警惕"过拟合"**：回测完美的策略，实盘可能因为是对历史的过拟合而失效。
