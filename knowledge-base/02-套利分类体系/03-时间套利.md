# 03 · 时间套利（Temporal Arbitrage）

> 同一份价值，在不同时间价格不同。低买高卖的"卖"和"买"发生在不同时点。

---

## 本质

现在便宜、未来贵（或反之）。通过"持有/等待/提前"跨越时间，赚取时点间的价差。时间套利的核心成本是**资金占用 + 持有风险**。

## 价差来源
- **时间摩擦**：未来需要等待，等待有机会成本（资金的时间价值）。
- **认知/偏好差异**：有人要现在的钱（折价卖），有人愿意等（吃溢价）。
- **周期性结构**：季节、节日、周期带来的可预测价格波动。

## 典型场景

| 形态 | 说明 |
|------|------|
| 囤货反季 | 反季买羽绒服/空调，旺季卖；丰收时囤、淡季卖 |
| 期现套利 | 期货与现货价差（含资金费率、基差交易） |
| 资产倒手 | 低位买入资产、持有、高位卖出（房产、收藏品、域名） |
| 预售/囤票 | 提前锁定低价，临近时高价卖（黄牛本质是时间+信息套利） |
| 加密资金费率 | 永续合约资金费率套利（持仓吃费率） |
| 折旧/翻新 | 买入贬值资产、修复后卖出 |

## 摩擦与风险
- **显性**：仓储费、资金利息、持有期的维护成本、保险。
- **隐性**：资金占用的机会成本、价格反向波动、需求预测失败、库存过期/损耗。
- **特有风险**：**时间越长，不确定性越大。** 你赌的是未来，而未来会变。最大的坑是"以为会涨，结果跌了/卖不掉"。

## 如何加护城河
1. **降低持有成本**：更便宜的仓储、更低的资金成本（这就是为什么有钱人/能低息融资的人更适合时间套利）。
2. **提升预测能力**：对周期、季节、需求的判断比别人准（认知优势）。
3. **对冲风险**：用衍生品锁定未来价格，把"赌方向"变成"赚确定价差"（这是期现套利的精髓——不赌涨跌，只赚基差）。
4. **周期化经营**：盯住"每年都会重复"的周期性价差，做成可重复的生意。

## 实操要点
- **区分"投机"和"套利"**：单纯囤货赌涨 = 投机（赌方向）。买现货同时卖期货锁定价差 = 套利（不赌方向）。**后者风险低得多，是真正的时间套利。**
- **资金成本是核心**：时间套利的年化 = 价差率 ÷ 持有时间。持有越久，年化越低。算账时一定要年化。
- **加密资金费率套利**：现货多 + 永续空（或反之），吃资金费率，是相对低风险的时间套利，但要管理好爆仓和费率反转风险。
