# 02 · 空间套利（Spatial Arbitrage）

> 同一份价值，在不同地点价格不同。最古老、最直观的套利——商业贸易的起点。

---

## 本质

价值在 A 地便宜、B 地贵，把它从 A 搬到 B，赚取差额。"丝绸之路"就是史上最大的空间套利项目。

## 价差来源
- **物理摩擦**（主要）：运输、仓储需要成本和时间，所以两地价格无法完全一致。
- **信息不对称**：A 地不知道 B 地的价格。
- **市场分割**：政策、关税、汇率、平台壁垒把市场隔开。

## 典型场景

| 领域 | 具体形态 |
|------|---------|
| 加密 CEX | 同一币种在币安/OKX/Coinbase 价差（跨所搬砖） |
| 加密 DEX | 同一 token 在不同链、不同 DEX、不同池子的价差 |
| 股票 | A股/H股/中概股 同公司不同市场的价差 |
| 实物 | 京沪白菜、产地直采 vs 城市零售、平行进口车 |
| 跨境 | 海淘代购、跨境电商、海外仓 |
| 电商平台 | 同一商品在淘宝/拼多多/京东/亚马逊价差 |

## 摩擦与风险
- **显性摩擦**：运费、关税、手续费、提现/转账费、跨链桥费。
- **隐性摩擦**：运输时间内的价格波动、汇率波动、损耗、库存积压。
- **特有风险**：搬运途中价差消失（尤其加密，转账确认期间价格已变）、对手方/平台风险（提不出币、账号被封）、合规风险（跨境、外汇）。

## 如何加护城河
1. **降低自己的搬运摩擦**：自建物流/海外仓、拿到更低手续费费率（大额 VIP 费率）、更快的跨链通道。别人因成本高做不了的价差，你能做。
2. **叠加信息维度**：比别人更早知道哪里有价差（信息前端）。
3. **变成通道**：从搬运者 → 物流商/聚合器/做市商，收"过路费"而非追价差。
4. **规模化资金调度**：多地预置库存/资金，免去"现搬"的时间摩擦（这是高频跨所套利的核心打法）。

## 实操要点
- **加密跨所**：核心是"两边预置资金"，不要真的来回转账（转账慢、价差早没了）。靠两边同时买卖 + 定期再平衡。手续费费率和提现速度是生死线。
- **跨境实物**：核心是物流和合规。务必走合法报关，灰关风险极高。
- **通用**：先算清"全摩擦成本"，很多空间价差看着大，扣完摩擦是负的。
