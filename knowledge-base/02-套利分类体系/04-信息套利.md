# 04 · 信息套利（Information Arbitrage）

> 一方知道，另一方不知道。中介、代理、经纪——所有靠"我知道你不知道"赚钱的角色。

---

## 本质

买卖双方信息不对称，你处在信息更充分的一方，靠"知道得更多/更早"赚取价差或撮合费。**这是离你"信息差"目标最直接相关的维度。**

## 价差来源
- **信息不对称**（纯）：典型且唯一的核心来源。
- 衍生：撮合价值（双方找不到彼此，你帮他们找到）。

## 典型场景

| 形态 | 说明 |
|------|------|
| 中介/经纪 | 房产中介、婚介、猎头、商务撮合——赚撮合费 |
| 代理/分销 | 拿到一手货源信息和渠道，转手给下游 |
| 二房东 | 信息+空间，整租散租，长租短租 |
| 一件代发 (Dropshipping) | 你知道哪里有便宜货源，买家不知道，你赚差价不囤货 |
| 比价/导购 | 知道全网最低价，帮人省钱赚佣金 |
| 信息服务/订阅 | 把"知道"直接打包卖（研报、情报、社群） |

## 摩擦与风险
- **显性**：获取信息的成本（调研、订阅、人脉维护）、撮合的沟通成本。
- **隐性**：**信息半衰期极短**——你知道的，很快别人也知道，价差迅速收敛。
- **特有风险**：被"去中介化"（买卖双方绕过你直接交易）、信息失真（你以为的内幕是假的）、信息变成大众常识后归零。

## 如何加护城河（信息套利最需要护城河，因为它最易被复制）
1. **持续的信息源优势**：不是一次性信息，而是一个"持续产出独家信息"的源头/网络。
2. **防止去中介化**：把交易闭环握在手里（如平台担保、不让双方直接见面、提供增值服务让双方离不开你）。
3. **从"卖信息"升级到"卖判断/能力"**：信息会过期，但"基于信息的判断力和执行力"难复制。
4. **规模化撮合 → 做平台**：当撮合量足够大，变成平台，靠网络效应建立壁垒。

## 实操要点
- **信息套利天然适合做成订阅/社群产品**（你提到的"市场套利机会订阅"就属于这类）。关键是持续产出"别人拿不到或来不及自己整理"的信息。
- **警惕：你能轻易拿到的信息，价差通常已经没了。** 真正值钱的是"加工后的判断"和"独家一手"。
- **去中介化是信息套利最大的死法**：设计商业模式时，第一考虑就是"怎么不被跳过"。
